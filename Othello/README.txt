Input coordenadas:
	x y
Énfasis, tiene que ser fila "espacio" columna, si esto no se hace el programa no garantiza el funcionamiento correcto.
No poner varias coordenadas seguidas, tienen que ir en formato:
x1 y1
x2 y2
etc..

Seguir estrictamente las instrucciones del programa respecto al input en los menús.

Per compilar localitzar-se al directori origen del projecte amb la terminal i executar:
make
Per executar el projecte executar:
make run
Per executar els drivers:
make run DriverIA
make run DriverEscenario
make run DriverCtrlDomain
make run DriverPartida
make run DriverTablero
